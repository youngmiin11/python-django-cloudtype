FAQ

  • 출석 안내 --> '오프라인 출석 안내'로 정정 합니다

    출석 기준(오프라인) 안내 입니다.


    1. 기준 시간 출석 동그라미

    2. 기준 시간 미출석 엑스

    3. 기준 시간 지각 세모


    * 온라인 출석은 지각이 없습니다.

       학습기간 내에 학습을 완료 하지 못하거나, 기간을 초과하여 완료 한 경우

       두가지 경우 모두 결석으로 처리되오니 유의 바랍니다. (학교 방침)

  • 동영상 재생에 대한 대처 방법

    안녕하세요.


    동영상 재생에 대한 대처 방법 안내드립니다.


    동영상 재생이 안되는 경우 익스플로러 초기화를 진행해 주시기 바랍니다.

  • Internet Explorer 업그레이드

    eClass 시스템에서는 다양한 브라우저를 지원하고 있으나 타 브라우저와의 호환성 문제로 

    MS의 지원이 종료된 Internet Explorer 에 대해서는 완벽한 지원이 힘듭니다.

    업데이트 된 브라우저를 사용해 주시길 권장합니다.

    eClass 시스템 사용 시에 문제가 있으실 경우 브라우저 업데이트로 문제가 해결될 수 있습니다.


    윈도우즈 8, 윈도우즈 8.1, 윈도우즈 10 을 사용하실 경우는 이미 Internet Explorer 11 버전이 설치되어 있으니

    아래의 작업이 필요하지 않습니다. 이 경우 eClass 시스템 사용에 문제가 있으시다면 은행 업무 및 사이트 이용 시

    설치된 액티브 X 프로그램이 문제를 일으킬 수 있으니 Chrome 브라우저로 eClass 시스템을 이용해

    문제가 일어나지 않는지 확인해 보시기 바랍니다. 


    Windows 7 버전을 사용하실 경우 아래의 링크를 클릭해 주십시오


    https://support.microsoft.com/ko-kr/help/17621/internet-explorer-downloads



    사용하고 계신 윈도우 제품 버전을 선택해주시면 아래와 같이 다운로드 버튼이 나타납니다.




    사용하고 계신 윈도우즈에 맞는 Internet Explorer 을 다운로드 합니다.


    다운로드 하신 파일을 더블 클릭하면 설치가 시작됩니다.


    설치가 완료되면 시스템을 다시 시작해야 하니 재시작하시기 전에 이용하고 계시던 프로그램을 정상 종료 해주십시오.



    참고: 윈도우즈 비트 확인법


    1. 좌측 하단 윈도우 시작 버튼을 선택해주세요. 시작 버튼을 클릭하여 메뉴가 열리면, 우측 중간아래 `제어판`이라는 항목을 선택합니다.



    2. 제어판 항목이 열리면 우측상단 아래 `보기 기준` 이라는 항목을 확인 하실수 있습니다. 여기서 보기기준을 큰 아이콘(L) 및 작은 아이콘(S)을 선택합니다. `큰 아이콘(L)`을 권장합니다.



    3. 제어판 항목에서 `시스템`이라는 항목을 선택합니다.



    4. 시스템 항목이 열리면 'windows 버전` 및 `시스템 종류`를 확인 하실수 있습니다. 시스템 종류라는 항목에 보이는 `XX비트 운영 체제`가 현재 사용중인 윈도우 운영체제 비트입니다.



  • Chrome 브라우저 설치하기

    eClass 시스템 사용 시에 문제가 있으실 경우 크롬 브라우저를 통해 문제가 해결될 수 있습니다.


    인터넷 브라우저의 주소창에 https://www.google.co.kr/chrome/browser/desktop/ 를 입력하시거나 링크를 클릭해 주십시오.




    화면 가운데의 파란색 버튼을 클릭하시면 크롬 브라우저의 다운로드가 시작됩니다.



    위와 같은 창이 뜨면 좌측 하단의 "동의 및 설치" 버튼을 클릭해 주십시오.



    다운로드 창이 뜨면 저장 위치를 지정해 주십시오.


    다운로드 하신 "ChromeSetup.exe" 파일을 더블 클릭 하시면 설치가 시작됩니다.


    설치가 완료되면 바탕화면에서 아래와 같은 아이콘을 찾아 클릭하시면 크롬 브라우저가 시작됩니다.



  • 동영상이 나오지 않을 경우 컴퓨터 설정
    ** 컴퓨터 환경이 맞지 않을 수 있으니 아래처럼 변경해 주시면 감사하겠습니다.
     
      기본값 복원
    [도구] => [인터넷옵션] => [고급]탭을 선택한 후 [기본값 복원] 버튼을 클릭한 다음
    [기본값 복원] 버튼을 클릭하고 [확인] 버튼을 클릭합니다.
    익스플로러 창을 닫은 다음 사이버 캠퍼스로 다시 접속합니다.



      보안설정
    [도구] => [인터넷옵션] => [보안]탭을 선택한 후 상단에 있는
    [보안 설정을 지정할 웹 컨텐트 영역을 선택하십시오] 영역에서 [신뢰할 수 있는 사이트]를 선택합니다.
    해당 페이지 하단 부분에 [이 영역에 적용할 보안 수준]에서 [기본 수준]을 선택한 후
    페이지 제일 아래 [적용] 버튼을 클릭합니다.


    크롬일 경우 Flash 기본값이 차단되어 있기 때문에 허용으로 변경해주셔야 합니다.
